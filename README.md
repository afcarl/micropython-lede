About
=====

A small micropython docker image based on LEDE.

Usage
=====

```
$ docker run -it zoobab/micropython-lede
MicroPython v1.8.6 on 2017-12-16; linux version
Use Ctrl-D to exit, Ctrl-E for paste mode
>>> 
```
